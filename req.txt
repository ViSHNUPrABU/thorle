Build a 100% config-driven dashboard UI in React+TypeScript using Vite.
This is an Observability platform like DataDog. UI can be inspired from DataDog.

1. Dashboard ( react-grid-layout ) ->
    * Draggable & droppable
2. ChartWidget ( A wrapper for echarts-for-react ). 
3. TableWidget ( A wrapper for mantine-react-table ).
4. DataWidget using plain React (no UI libraries) to display info.
5. WidgetWrapper -> 
    * APIService 
    * ContextService ( which helps to share data between widgets in dash )
    * Loading, Empty, Error template to display after API
    * Visibilty - some condition to display the widget
6. NavBar ( no lib ) ->
    * Sidebar
    * Initially should have 'Dashboards'
7. Every navigation should be routed for reloading and link sharing.
8. Configs should have types.
9. Dashboard, NavBar, ChartWidget, TableWidget, DataWidget, WidgetWrapper should be 100% fully
    config driven.
10. Only use these 3 libraries and ask before including a new one and mention it's usecase.
    It should have less bundle size.
11. Everything should be 100% config driven. Make it modular, extensible, performant, organised.
